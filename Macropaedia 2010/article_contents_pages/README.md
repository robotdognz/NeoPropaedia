# Article Contents-Page Photos

Store article contents-page photos for the 2010 data project here.

Recommended naming:

- `v13-p0302-contents.jpg`
- `v13-p0302-contents-01.jpg`
- `v13-p0302-contents-02.jpg`

Where:

- `v13` is the Macropaedia volume number
- `p0302` is the article start page label without punctuation

These photos will later support:

- subsection mapping review
- Propaedia mapping reconstruction
- Britannica website breakdown matching
