# Volume Contents Photos

- `macropaedia-2010-volume-01-contents.jpg`
- `macropaedia-2010-volume-02-contents.jpg`
- ...
- `macropaedia-2010-volume-17-contents.jpg`

These are the seed images for the OCR and article metadata pipeline.
