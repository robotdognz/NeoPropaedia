# Macropaedia 2010 Image Root

Keep all 2010 Britannica photo evidence for this project under this root:

- volume contents photos
- Propaedia page photos
- article contents-page photos

The project database stores image paths relative to this folder.

Current status:

- the original 17 volume contents photos now live in `contents_pages/`

Recommended future subfolders:

- `contents_pages/`
- `propaedia_pages/`
- `article_contents_pages/`

Recommended naming:

- Propaedia pages: `propaedia-page-0123.jpg`
- Article contents pages: `v13-p0302-contents.jpg`

If one logical page needs multiple photos, add a suffix:

- `v13-p0302-contents-01.jpg`
- `v13-p0302-contents-02.jpg`

Try to keep filenames stable once they have been referenced in the database.
